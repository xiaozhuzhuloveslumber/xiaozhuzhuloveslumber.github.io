啊..终于又重新弄好了因为前几个小时手贱删了原来的blog文件夹
好在第二次弄轻车熟路，，没用太久时间
以下是原博文的复述:
本博客采用的是Hexo+GitHub page的方案家境贫寒，莫得自己的服务器
我的搭建参考了这两篇文章：
如何搭建个人博客（详细图解）
超详细 Hexo+ Github Page搭建技术博客教程
这两篇文章已经讲得很详细，相信你自己搭建的话不会遇到太大的问题
以下我遇到的文章里面没有涉及的报错：
1.在首次安装hexo的时候，，可能会卡死在过程中。这时要先运行 

npm config set registry https://registry.npm.taobao.org

修改npm的安装源后就可以正常安装hexo了
2.在配置博客的config部署到GitHub上时..要注意type后面有一对单引号..git要写在里面（是的我当时就以为是搞错了就擅自删去了）
3.再有就是各位弹幕大神的援助了.. .

hmm..我遇到的就这些了吧

就自己是闲的出屁而且刚好看见了同学博客写了搭建的方法..那就试试wei。 于是这个blog就诞生了！！学到了不少，也是一次很有价值的尝试呢！
当然..像我这种电脑小白..全部都是现学现用，，过程确实没有那么愉快..各种报错不知道怎么搞的时候真的是好想
